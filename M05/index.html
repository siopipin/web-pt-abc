Ini Halaman Web Saya

Semuanya sedang berusahan untuk membuat halaman web yang baik dan tentunya kita sebagai pengguna harus bisa menggunakan web tersebut menjadi lebih baik dan terus lebih baik dari sebelumnya. Oleh karena itu, kita harus bisa membuat tampilan web dengan menerapakan CSS dan bila perlu kita juga perlu menambahkan javascript untuk membuat sedikit animasi ke dalam teks tersebut.
